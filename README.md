# Gitbook学习笔记

GitBook是一个基于 Node.js 的命令行工具，可使用 Github/Git和Markdown来制作精美的电子书。

这是个人学习GitBook的笔记，请点击下面的链接阅读或者下载电子版本:

- 在线阅读
	- [国外服务器][gitbook]：gitbook提供的托管，服务器在国外，速度比较慢，偶尔被墙，HTTPS
	- [国内服务器][skyao]：腾讯云加速，国内网速极快，非HTTPS

- [下载mobi格式][mobi]
